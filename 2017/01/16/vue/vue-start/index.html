Vue-start

发表于 2017-01-16   |   分类于 vue

Vue是一个构建用户界面的渐进式框架，不像其他庞大的框架一样，vue是自底向上合理增长的开发设计。
它的核心库只专注于视图层，而且非常容易去理解，并且可以和其他库或者已存在的项目进行结合。从另一方面来说，Vue也完全有能力去驱动复杂的被使用在由流行工具和支持的库所结合的单页面应用。

声明式渲染

Vuejs的核心就是能让我们能用简单直接的语法模板声明式的渲染数据
示例中包含html和js两部分，html部分建立一个id为app的div，里面的内容用两个大括号括起来，js中vue组件包括el属性和data属性，el属性值为绑定需要操作的DOM的id，并且需要#号，data属性顾名思义是数据，用大括号括起来，data属性里面是一对对键值对，键是div中的内容名，值需要用单引号括起来。

这个算是完成了我们第一个vue app了，这个是最简单的方式，看起来和仅仅渲染一个字符串模板非常相似，但是Vue在后面看不见的地方做了很多工作。数据和DOM被连接了起来，并且所有的东西都是响应式的，这点可以通过打开浏览器在控制台中给app.message一个不同的值来验证。

在这个实例中可以看到一个新的东西，v-bind这个属性被称为指令，指令都有一个v-的前缀来指明它们是Vue提供的特殊属性，并且你可能已经猜到了，它们向渲染的DOM提供特殊的响应式行为，基本在说：保持这个元素的title属性依靠Vue实例中的message属性的值而更新变化。
如果你打开了你的JavaScript控制台，并且输入：app2.message = ‘some new message’，就可以看到绑定的html在这种情况下title属性会被更新。
