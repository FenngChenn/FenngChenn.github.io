react---DOM Event Listeners in a Component

发表于 2017-04-15   |   分类于 react

此条目展现了怎样联系到一个不是由React提供的DOM事件，对于其他库比如jQuery是一个很好的集成。

componentDidMount在组件加载完成之后被调用，并且拥有DOM的表现。这通常是一个你可以联系到一般DOM事件的地方。
注意事件回调被绑定到了react组件而不是原始元素，对于你来说，React会通过一个自动绑定过程来自动绑定方法到现有组件实例中去。

如果使用ES6的话，那么遵循ES6 class的语义，就不会有自动绑定this到实例上，必须显式的用.bind(this)或者箭头函数=>

建议在构造函数中绑定事件处理器，这样对于所有实例只需绑定一次
